# 第一卷：离乡

## 第8章 房东老刘

老刘这个人，在北京住了一辈子，哪都没去过。

"去外面干啥？哪儿有北京好。"这是他最爱说的一句话。

他是土生土长的北京人，祖上三代都在西苑这片地界上。年轻的时候在国营工厂上班，后来工厂倒闭了，他就把自家院子翻盖成了三层的自建房，隔成十几个小间，当起了包租公。

每个月收租的时候，他叼着烟，挨家挨户地敲门，手里拿着一个皱巴巴的小本子，收了钱在上面打个勾。他的字写得很难看，歪歪扭扭的，但他记得特别清楚——谁交了谁没交，谁欠了多少钱，全在他的脑子里。

老刘有他自己的规矩。第一，不赊账。房租到日子就得交，晚一天加十块。第二，不准带外人过夜。第三，不准在屋里做饭——虽然大家都没遵守，用电饭煲偷偷煮饭的大有人在，他也就睁一只眼闭一只眼。第四，也是最奇怪的一条：不准吵架。

"打架我不管，但别在我这院子里吵架。"有一次他对一个新来的租客说，"听着闹心。"

新租客是一个湖南来的小伙子，二十出头，在餐馆当服务员。他问老刘为什么。老刘没回答，只是摆摆手，说了一句："不想听。"

后来我才知道为什么。老刘年轻的时候，和他老婆吵了一辈子的架。后来他老婆得了癌症，走了。临走那天，他们还在为一件小事拌嘴。她走了之后，老刘就再也听不得人吵架了。

这些话不是老刘自己说的。是巷子口卖煎饼的河北大姐告诉我的。河北大姐在附近卖了六年煎饼，对这里的人和事门儿清。

"老刘啊，是个好人。"她一边摊煎饼一边说，"你别看他凶，他从来不为难人。去年有个租客实在没钱交房租，他让人家欠了三个月，最后人家走了，钱也没要回来。他嘴上骂了两天，第三天就忘了。"

"那他为什么不承认？"

"北京人嘛，嘴硬心软。"她笑了笑，"你慢慢就知道了。"

有一天晚上，我下班回来，发现门锁坏了——钥匙插进去，转不动。我试了好几次，急得满头大汗。老刘正好在院子里，走过来看了看，没说话，转身进屋拿了一个工具箱。他蹲下来，用手电筒照着锁眼，捣鼓了二十分钟。夏天晚上闷热，他额头上全是汗，烟灰掉了一地。

"好了。"他站起来，把钥匙递给我，"锁芯里面锈了，上了点油。这破房子，啥都旧了。"

"谢谢刘叔。"

"谢啥谢。"他擦了擦汗，点了一根烟。"住着吧，有啥事跟我说。"然后他又加了一句："别扛着。"

这三个字，他说的很轻，像是在自言自语。但我听见了。那天晚上，我躺在床上想了很久。老刘说的"别扛着"，不是"有啥事跟我说"。他是在说——我知道你扛着很多事，但是别一个人扛。

后来我才慢慢明白，老刘见过的北漂，比我认识的还多。来的时候都是年轻人，走的时候各有各的原因。有的赚了钱走了，有的没赚到钱也走了，有的在这结婚生子落地生根，有的混不下去回了老家，再也没有回来。老刘见过太多人。所以他不问，也不劝。他只是收他的房租，修他的门锁，偶尔在你最需要的时候，说一句不轻不重的话。

很多年以后，我回西苑看他的时候，他已经老了。院子还是那个院子，但住的人全换了。我问他，还记得我吗。他看了我半天，说："记得。就是那个住地下室的，瘦得跟猴似的。"然后他点了一根烟，笑了笑。"现在胖了。好啊，胖了好。"
